# Grocery App

This is a simple grocery app built with FastAPI.

## Setup

1. Install dependencies:
    ```bash
    python3 -m venv venv
    source venv/bin/activate
    pip install -r requirements.txt
    ```

2. Run the application:
    ```bash
    fastapi dev
    ```

## Endpoints

- `POST /users/`: Create user
- `POST /users/me`: Get current user
